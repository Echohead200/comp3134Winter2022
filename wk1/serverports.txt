Q: What is the definition of server port?
A: server ports are the logical endpoints of a network connection
that is used to exchange information between a web server and a web client

Q:What is server port range?
A: 0 to 65535
Q:How many bits does number above represent?
A: 16 bits
Q:State 10 common server ports
A: Ports 20 and 21: File Transfer Protocol 
Port 22: Secure Shell (SSH)
Port 25: Simple Mail Transfer Protocol (SMTP)
Port 53: Domain Name System (DNS)
Port 80: Hypertext Transfer Protocol (HTTP)
Port 123: Network Time Protocol (NTP)
Port 179: Border Gateway Protocol (BGP)
Port 443: HTTP Secure (HTTPS)
Port 500: Internet Security Association
Port 3389: Remote Desktop Protocol
